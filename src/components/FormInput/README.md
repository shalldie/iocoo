## 表单输入框组件

### props

| 名称   |  类型  | 是否必填 | 默认值 | 说明                               |
| :----- | :----: | :------: | :----- | ---------------------------------- |
| inputIndex    | Number |    否    | 0      | 子组件唯一标识，如果循环输出则必填 |
| lbl    | String |    是    | field  | input 的label                      |
| defVal |  any   |    否    |        | input 默认值                       |


### Events

| 名称   | 说明                              | 回调参数          |
| :----- | --------------------------------- | ----------------- |
| update | input 的 value 发生改变的时候触发 | 2个，input的value ,组件初始化时传入的inputIndex|
